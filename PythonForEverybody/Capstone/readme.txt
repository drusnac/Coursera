The contains the capstone files for the course. The project was to identify a data source somewhere online, sanitize and store it in a sqlite db.
Once the data was available some sort of data vizualization would be used to show the data.

I pulled my data from https://www.start.umd.edu/gtd/contact/ which has an xlsx file that contains the terror attacks around the world since
1970.

My goal was to create a heatmap, showing which countires are hardest hit by terror.
